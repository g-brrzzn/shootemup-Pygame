### Installation

```console
# clone the repo
$ git clone https://github.com/g-brrzzn/shootemup-Pygame

# change the working directory
$ cd shootemup-Pygame

# install the requirements
$ python -m pip install -r requirements.txt

# run the Game.py
$ python Game.py
```
